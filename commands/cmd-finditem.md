# /finditem

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/finditem
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the "Find Item Window", allowing searches through your: bank, equipped items, inventory, real estate, item overflow and parcel. Note: This will not search your Dragon's Hoard.
<!--cmd-desc-end-->
