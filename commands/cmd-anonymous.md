# /anonymous

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/anonymous [on|off]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggle. When on, prevents others from seeing your class, level, race, guild, and current zone through [/who](cmd-who.md) and [/who all](cmd-who.md) queries. Common abbreviation: `/anon`
<!--cmd-desc-end-->
