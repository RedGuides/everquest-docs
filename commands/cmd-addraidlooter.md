# /addraidlooter

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/addraidlooter <name>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Add your current target, or specified character name, to the list of valid looters in the raid.
