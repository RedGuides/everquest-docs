# /overseermassconvert

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/overseermassconvert <rarity> <keep_count>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Converts duplicate agents of a specified rarity, leaving a specified number of duplicates unconverted with a minimum of 1 remaining.
<!--cmd-desc-end-->
