# /memspellset

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/memspellset {index|<name>}
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Loads the specified spellset index or name. Spell sets are created with the spell gems window.
<!--cmd-desc-end-->

!!! example
    `/memspellset 1` - Loads the first spell set in your spell set list.
