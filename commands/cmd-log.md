# /log

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/log [on|off]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles output to `Logs\eqlog_<name>_<server>.txt`. The logging state persists across logins.
<!--cmd-desc-end-->
