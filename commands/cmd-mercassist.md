# /mercassist

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/mercassist [on|off]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Calls for your mercenary to assist when auto-assist is not activated. To automatically assist, provide the `on` option.
<!--cmd-desc-end-->
