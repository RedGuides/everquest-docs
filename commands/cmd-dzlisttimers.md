# /dzlisttimers

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/dzlisttimers
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
List any outstanding replay timers you have for all expeditions. This is the amount of time you must wait before being allowed to enter another instance of that zone.
<!--cmd-desc-end-->
