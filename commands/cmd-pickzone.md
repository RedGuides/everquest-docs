# /pickzone

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/pickzone [fewestplayers]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the Pick Zone window if available at the moment in the current zone.
<!--cmd-desc-end-->

## Options

**`fewestplayers`**
:   Automatically picks the zone instance with the lowest number of players.
