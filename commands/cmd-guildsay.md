# /guildsay

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/guildsay <text>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Sends `<text>` to guild chat. Seen by all in your guild. Common abbreviation: `/gu`
<!--cmd-desc-end-->
