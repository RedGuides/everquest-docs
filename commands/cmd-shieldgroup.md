# /shieldgroup

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/shieldgroup
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Allows a Warrior to shield his group from damage, provided they have the group shielding spell affect.
