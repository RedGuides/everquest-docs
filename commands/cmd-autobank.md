# /autobank

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/autobank
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
With the Bank window open and with an item on your cursor, will attempt to place the item into a spare bank slot.
<!--cmd-desc-end-->
