# /gsay

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/gsay <text>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Sends `<text>` to group chat. Seen by all in your group. Common abbreviation: `/g`
