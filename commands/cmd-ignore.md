# /ignore

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/ignore [<name>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Lists the names currently being ignored. If a player `<name>` is supplied, it will toggle their inclusion on the list; a new name will be added, an existing name removed.
<!--cmd-desc-end-->
