# /taskoverlay

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/taskoverlay
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles the "Task Overlay" window, which gives a summary of quests and your progress.
<!--cmd-desc-end-->
