# /report

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/report <playername>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Snitch on `<playername>`. The first 20 lines of visible text in each chat window will be attached to the report. Scroll to display the text you wish to report.
<!--cmd-desc-end-->
