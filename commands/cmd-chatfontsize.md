# /chatfontsize

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/chatfontsize <0-10>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Adjust the size of the font in the chat window, from size 0 to size 10, with 0 being the smallest. 3 is default.
<!--cmd-desc-end-->

!!! note
    - `9` seems to show a different font.
