# /removeraidlooter

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/removeraidlooter <name>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Removes the specified character `<name>` from the list of players permitted to loot on the raid.
<!--cmd-desc-end-->
