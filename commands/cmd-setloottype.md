# /setloottype

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/setloottype <1-3>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Sets who can loot npc corpses during a raid: 1=Raid Leader, 2=Raid Leader and Group Leaders, 3=Raid Leader and Selected Raid Looters (used in conjunction with [/addraidlooter](cmd-addraidlooter.md))
<!--cmd-desc-end-->
