# /stand

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/stand
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Makes your character stand up from a crouch or sit position. Unlike [/sit](cmd-sit.md), it's not a toggle.
<!--cmd-desc-end-->
