# /setstartcity

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/setstartcity [<ID>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Set your starting city. Can be used once by some characters with multiple starting cities or those whose starting city was reset accidentally.
