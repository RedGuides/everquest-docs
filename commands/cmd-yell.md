# /yell

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/yell
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Unlock all *Encounter Locked NPCs* that you're able to. One second timer. Also displays, 'You yell for help.' People around you see "XXX yells for help from `<direction>`".
<!--cmd-desc-end-->
