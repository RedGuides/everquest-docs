# /duel

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/duel
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Initiates a duel between you and the currently targeted player. Duel is in effect until one "dies" (is knocked unconscious) or flees the zone.
<!--cmd-desc-end-->
