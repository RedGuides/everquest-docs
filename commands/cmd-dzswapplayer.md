# /dzswapplayer

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/dzswapplayer <character to remove> <character to add>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Swap out one player for another. Must be leader. The player to be added has to approve, or the command fails.
<!--cmd-desc-end-->
