# /loginterval

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/loginterval [seconds]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Sets the log file write interval. A value of 0 will write immediately to the file without any delay. The default log file write interval is 1 second(s).
<!--cmd-desc-end-->
