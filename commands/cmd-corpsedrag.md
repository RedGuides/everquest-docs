# /corpsedrag

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/corpsedrag
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Will auto-drag your current targeted corpse, if permitted. You can drag up to two corpses.
