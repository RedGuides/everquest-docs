# /cast

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/cast <slot# | spellname>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Casts the spell in the specified gem slot (1-10) or by the name of a memorized spell. Partial spell names are accepted.
<!--cmd-desc-end-->
