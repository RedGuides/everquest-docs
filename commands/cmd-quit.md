# /quit

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/quit
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Immediately closes the game. The full command name must be used. This command is functionally identical to [/exit](cmd-exit.md).
<!--cmd-desc-end-->
