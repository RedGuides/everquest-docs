# /keyrings

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/keyrings
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the "Key Rings" window. Here you can see your Mounts, Illusions, Familiars, Hero's Forge and Teleportation items.
<!--cmd-desc-end-->
