# /keyrings

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/keyrings [<tab>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the Key Rings window. If a tab name is specified, that tab is displayed.
<!--cmd-desc-end-->

## Examples

!!! example
    `/keyring mount` - Opens the mount tab.
