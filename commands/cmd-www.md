# /www

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/www <website>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the default browser to the given web page. Unlike [/url](cmd-url.md), this can only open web pages.
<!--cmd-desc-end-->
