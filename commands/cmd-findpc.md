# /findpc

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/findpc <name>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Used to show a yellow path toward a player character. The player must be visible in the find window, such as when they're in your group or raid.
<!--cmd-desc-end-->
