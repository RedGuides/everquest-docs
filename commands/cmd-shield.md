# /shield

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/shield <playername>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Allows Warriors to intercept part of the damage intended for another. If no `<name>` supplied, will use target.
<!--cmd-desc-end-->
