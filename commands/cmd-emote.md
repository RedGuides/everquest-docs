# /emote

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/emote <text>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Emotes the `<text>`. Alternate commands: `/me`, `:`
<!--cmd-desc-end-->

## Examples

!!! example "`/em wipes the gore from his axe!` - Will display as "Player wipes the gore from his axe!""
