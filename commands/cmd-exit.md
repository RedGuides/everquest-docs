# /exit

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/exit
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Immediately exit you from the game. 
Note: your character will still be in an active state, meaning they can still die, until the server logs them out a few minutes later.
<!--cmd-desc-end-->
