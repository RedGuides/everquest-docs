# /roleplay

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/roleplay [on|off]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Anonymizes information from [/who](cmd-who.md), but unlike [/anon](cmd-anonymous.md), your guild tag is still displayed.
