# /gmarknpc

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/gmarknpc <1-3>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Mark an NPC for your group with a number from 1 to 3.  Alternate: `/marknpc`
