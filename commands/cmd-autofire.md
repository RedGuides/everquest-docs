# /autofire

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/autofire [on|off]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggle automatic firing of your ranged weapon. Using this command will cause the Player window to flash blue. A target is not required when disabling autofire. Note: Turning on [/attack](cmd-attack.md) will turn off autofire and switch to melee.
<!--cmd-desc-end-->
