# /changefamiliar

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/changefamiliar
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the familiar name change window, allowing you to rename your familiars. Requires the Potion of Familiar's Amnesia item from the marketplace.
<!--cmd-desc-end-->
