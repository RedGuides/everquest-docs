# /calendar

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/calendar
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the in-game Calendar window. The event calendar lists all active events; Title, date (from/to), time (from/to) and brief description. All dates and times are displayed in your local time zone.
<!--cmd-desc-end-->
