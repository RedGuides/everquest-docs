# /offlinemode

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/offlinemode
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
When in [Trader](cmd-trader.md) or [Buyer](cmd-buyer.md) mode, this command will log you off but leave your character in-game trading.
<!--cmd-desc-end-->
