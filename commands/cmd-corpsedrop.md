# /corpsedrop

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/corpsedrop
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Used to stop dragging a corpse that you're holding with [/corpsedrag](cmd-corpsedrag.md). Will use currently targeted corpse if selected.
<!--cmd-desc-end-->
