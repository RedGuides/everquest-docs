# /note

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/note [<text>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Opens the "Notes Window" unless optional `<text>` is provided. Any `<text>` provided will be date/time stamped. The information is saved in the file notes.txt and is available across all characters on an account.
<!--cmd-desc-end-->
