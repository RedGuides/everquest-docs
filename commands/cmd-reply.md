# /reply

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/reply <text>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Reply to the latest received [/tell](cmd-tell.md). Can also bind replies to a key, "R" by default, and then cycle through names with "Tab".
<!--cmd-desc-end-->
