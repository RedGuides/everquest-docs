# /servertransfer

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/servertransfer
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles the Server Transfer List window. Some transfers are free, others will require a token from the [marketplace](cmd-marketplace.md).
