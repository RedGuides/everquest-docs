# /invite

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/invite [<name>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Invites your current target or supplied player name to the group. Can also be used to accept an invitation.
<!--cmd-desc-end-->
