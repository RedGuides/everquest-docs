# /auction

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/auction <message>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Message the auction channel in the current zone. 
Note: On the detail window of an item, you can click on the item's icon to make a clickable-link in your messages.
<!--cmd-desc-end-->
