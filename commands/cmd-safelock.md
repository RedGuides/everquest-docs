# /safelock

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/safelock <password>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggle. Locks the interface with `<password>`. To unlock the interface, repeat the command with the same password. The full command must be typed to function. Not case-sensitive.
<!--cmd-desc-end-->
