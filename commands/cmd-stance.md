# /stance

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/stance [ Aggressive | Balanced | Burn | Burn AE | Efficient | Passive | Reactive ]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
