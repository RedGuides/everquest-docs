# /target

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/target [<name> | group <1-6>]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Targets `<name>` or group member `<#>`.
<!--cmd-desc-end-->

## Options

**name**
:   Name to target

**group `<#>`**
:   Target group member 1-6. Similar to the default keys F1 - F6.
    !!! example "`/target group 3` - targets group member 3."
