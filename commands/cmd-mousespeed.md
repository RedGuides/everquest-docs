# /mousespeed

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/mousespeed <multiplier>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Increase the speed of the mouse cursor in multiples of the amount specified. Default multiplier is 0, use 0 or 1 for default, 2 = double the speed, 3 = triple, etc.
<!--cmd-desc-end-->
