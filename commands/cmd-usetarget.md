# /usetarget

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/usetarget
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Simulates a 'right-click' on a targeted NPC, or left click on targeted player. Useful when merchants are covered in other players, making them hard to click.
<!--cmd-desc-end-->
